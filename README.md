## Enviroment Variables

mark to represent git status

| name                 | mark (default) |
|----------------------|----------------|
| `HINA_GIT_MODIFIED`  | `*`            |
| `HINA_GIT_ADDED`     | `+`            |
| `HINA_GIT_DELETED`   | `-`            |
| `HINA_GIT_COPIED`    | `$`            |
| `HINA_GIT_RENAMED`   | `~`            |
| `HINA_GIT_UNMERGED`  | `=`            |
| `HINA_GIT_UNTRACKED` | `%`            |
